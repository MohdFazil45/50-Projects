Day 17 - Custom Range Slider
A sleek custom-styled range slider that demonstrates how to replace the default browser slider UI with a more modern and visually appealing design.

##  What I Learned

In CSS

1 How to style `<input type="range">` using:  `::-webkit-slider-thumb` and `::-webkit-slider-runnable-track` for Chrome/Safari,`::-moz-range-thumb` and `::-moz-range-track` for Firefox

2 Use of `-webkit-appearance: none` to override default styles
