Day 14 - Notes App 
A simple Notes App built using HTML, CSS, and JavaScript that allows users to add, edit, delete, and persist notes in the browser using localStorage. The app also supports Markdown formatting using the marked library.

## What I Learned 

In JavaScript 

1 Use of `localStorage` and `sessionStorage` to persist data in the browser.

2 How to update localStorage in real-time on text input changes.

3 How to dynamically create and manage DOM elements using `createElement`, `innerHTML`, `classList`, and     `appendChild`.

4 How to convert Markdown to HTML using the `marked.parse()` function.
